# Desafio

Você foi contratado para desenvolver um sistema de recomendação de músicas que utiliza grafos para identificar padrões de escuta e sugerir novas faixas aos usuários. O sistema deve ser capaz de:

- Representar usuários, músicas, artistas e gêneros como nós de um grafo
- Representar interações (como escutar, curtir, seguir) como arestas com prioridade
- Utilizar consultas Cypher para gerar recomendações personalizadas com base em conexões do grafo
